#### 1. 链接锚文本参与计数（不含 URL）

- 处理逻辑：匹配 `[锚文本](URL)` 格式，提取 `[]` 内的锚文本（英文不计数），删除 `()` 内的 URL；
- 示例：`[Git教程](https://git-scm.com)` → 保留「教程」计数中文 2 字 。

#### 2. 图片中文说明计数，英文说明不计

- 处理逻辑：匹配 `![图片说明](URL)` 格式，仅提取说明中的中文字符，过滤英文、数字、符号；
- 示例：`![Python代码示例](code.png)` → 仅保留「代码示例」（2 字），英文「Python」不计入；`![截图2024](img.jpg)` → 保留「截图」（2 字），数字「2024」不计入。

#### 3. 时间戳、Hash 值不参与计数

- 时间戳过滤：覆盖常见格式（`2024-10-01`、`14:30:00`、`2024/10/01`、`2024年10月1日`）；
- Hash 值过滤：匹配 32-64 位十六进制字符串（常见于 Git 提交号、文件校验码，如 `a1b2c3d4e5f6...`）。

#### 4. 额外优化（提升实用性）

- 自动识别脚本所在目录：无需手动切换工作目录
- 递归扫描子文件夹：支持扫描文件夹下所有子目录的 `.md` 文件；
- 错误捕获：读取文件失败时（如权限问题、编码异常）会提示原因，不中断整体统计

## 脚本说明
本 Python 脚本（count_md_words.py）用于计算 Markdown 文档中有意义的文本字数。已经自动去除代码块、去除 Markdown 格式、去无意义字符。

* 去除（标题 #、粗体 **、链接等）和代码块（```/` 包裹）。
* 中文计数、英文不计数。仅提取锚文本中的中文字符，英文 / 数字 / 符号全部过滤。
* 无意义字符（空格、换行、符号）不计入。

可以批量计算当前脚本所在文件夹下所有子目录的 `.md` 文件字数。

### 脚本使用方法

可以选择以下两种运行方法。注意要将当前脚本文件与需要翻译的 md 文件夹（可以有很多子文件夹）放在同一目录下。

#### 一、使用终端运行 Python 脚本

* 打开终端（Windows：cmd；Mac：Terminal），通过 cd 命令进入 Markdown 文档所在文件夹（例：cd C:/Docs/Markdown）。
* 输入命令 python count_md_words.py 并回车，即可批量统计所有 .md 文件的有效字数。

#### 二、使用 VS Code 插件运行 Python 脚本（推荐）

若需要修改脚本后立即运行，推荐用 VS Code。

1. 安装 VS Code：官网 https://code.visualstudio.com/ 下载，拖拽到「应用程序」即可。
2. 安装 Python 插件：打开 VS Code，左侧点击「扩展」，搜索 “Python”（作者：Microsoft），点击「安装」。
3. 打开脚本文件：
   - 点击 VS Code 左上角「文件」→「打开文件夹」，选择脚本所在文件夹；
   - 在左侧文件列表中，点击脚本文件（如 `count_md_words.py`）打开。
4. 运行脚本：
   - 点击脚本编辑区右上角的 「运行 Python 文件」按钮；
   - 或右键点击编辑区，选择「在终端中运行 Python 文件」；
   - 运行结果会显示在底部「终端」面板中。

### 脚本中计数细节说明

#### 1. 链接锚文本参与计数（不含 URL）

- 处理逻辑：匹配 `[锚文本](URL)` 格式，提取 `[]` 内的锚文本（英文不计数），删除 `()` 内的 URL；
- 示例：`[Git教程](https://git-scm.com)` → 保留「教程」计数中文 2 字 。

#### 2. 图片中文说明计数，英文说明不计

- 处理逻辑：匹配 `![图片说明](URL)` 格式，仅提取说明中的中文字符，过滤英文、数字、符号；
- 示例：`![Python代码示例](code.png)` → 仅保留「代码示例」（2 字），英文「Python」不计入；`![截图2024](img.jpg)` → 保留「截图」（2 字），数字「2024」不计入。

#### 3. 时间戳、Hash 值不参与计数

- 时间戳过滤：覆盖常见格式（`2024-10-01`、`14:30:00`、`2024/10/01`、`2024年10月1日`）；
- Hash 值过滤：匹配 32-64 位十六进制字符串（常见于 Git 提交号、文件校验码，如 `a1b2c3d4e5f6...`）。

#### 4. 额外优化（提升实用性）

- 自动识别脚本所在目录：无需手动切换工作目录
- 递归扫描子文件夹：支持扫描文件夹下所有子目录的 `.md` 文件；
- 错误捕获：读取文件失败时（如权限问题、编码异常）会提示原因，不中断整体统计

## 脚本说明
本 Python 脚本（count_md_words.py）用于计算 Markdown 文档中有意义的文本字数。已经自动去除代码块、去除 Markdown 格式、去无意义字符。

可以批量计算当前脚本所在文件夹下所有子目录的 `.md` 文件字数。

### 脚本使用方法

可以选择以下两种运行方法。注意要将当前脚本文件与需要翻译的 md 文件夹（可以有很多子文件夹）放在同一目录下。

#### 一、使用终端运行 Python 脚本

* 打开终端（Windows：cmd；Mac：Terminal），通过 cd 命令进入 Markdown 文档所在文件夹（例：cd C:/Docs/Markdown）。
* 输入命令 python count_md_words.py 并回车，即可批量统计所有 .md 文件的有效字数。

#### 二、使用 VS Code 插件运行 Python 脚本（推荐）

若需要修改脚本后立即运行，推荐用 VS Code。

1. 安装 VS Code：官网 https://code.visualstudio.com/ 下载，拖拽到「应用程序」即可。
2. 安装 Python 插件：打开 VS Code，左侧点击「扩展」，搜索 “Python”（作者：Microsoft），点击「安装」。
3. 打开脚本文件：
   - 点击 VS Code 左上角「文件」→「打开文件夹」，选择脚本所在文件夹；
   - 在左侧文件列表中，点击脚本文件（如 `count_md_words.py`）打开。
4. 运行脚本：
   - 点击脚本编辑区右上角的 「运行 Python 文件」按钮；
   - 或右键点击编辑区，选择「在终端中运行 Python 文件」；
   - 运行结果会显示在底部「终端」面板中。

### 统计规则明细

#### 计入字数的内容

1. 普通正文：
   - 中文汉字（如 “教程、代码”）；
   - 中文标点符号（全角，如。，！？；：""''（）【】、《》等）；
   - 所有数字（不含时间戳 / Hash 中的数字，如 “123、4567”）。
2. 链接元素：`[锚文本](URL)` 格式中 `[]` 内的中文 + 中文标点（英文锚文本不计入，URL 完全剔除）。
3. 图片元素：`![图片说明](URL)` 格式中 `[]` 内的中文 + 中文标点（英文说明不计入，图片链接完全剔除）。
4. 代码内容：代码块（```包裹）、行内代码（` 包裹）中的中文、中文标点及数字（代码注释、字符串中的中文 / 标点均计入）。

#### 不计入字数的内容

1. 格式相关：Markdown 标题 #、粗体 / 斜体 *_、分割线 ---、引用 >、表格分隔符 | 等格式符号；
2. 链接 / URL：所有 `()` 内的 URL 地址（如 `https://xxx.com`）、图片链接地址；
3. 语言相关：所有英文字母（含纯英文单词、代码中的英文关键字 / 变量名，如 “python、print”）；
4. 特殊内容：
   - 时间戳（如 `2024-10-01`、`14:30:00`、`2024年10月1日`、`20241001143000`）；
   - 32-64 位 Hash 值（如 `a1b2c3d4e5f6...`，含其中数字和字母）；
5. 列表前缀：列表前的数字 + 符号（如 `1.`、`2、`）。
